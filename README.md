# GameMania

This is an Expo & React Native Typescript & Firebase project that displays games and allows users to interact with them.

## Overview

When you open the app it opens with Login screen and after that it welcomes you with Home screen that displays 20 popular games fetched from RAWG API. When you press one of the cards it takes you to game detail screen where you can add that game to your favorites. In the second tab you can search every game that is in the RAWG API and in the last tab you can see your favorite games displayed and finally you can log out from the app.
